# unicorn-name-generator-uipath
UiPath RPA project that automates a web-based unicorn name generator using form input, selection, and data extraction.
Unicorn Name Generator – UiPath RPA
Tool: UiPath Studio
Description:
This automation interacts with a web application to generate a unicorn name based on user inputs. The bot enters required details, submits the form, and extracts the generated unicorn name automatically.
Process Flow:
Open web application
Enter user name
Select birth month
Click “Get Name”
Extract and display unicorn name
Activities Used:
Type Into
Click
Select Item
Get Text
Web Automation activities
Deployment:
Published as a solution and successfully activated in UiPath Orchestrator.
Status:
Successfully executed without errors.
